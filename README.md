## 使用方法

本项目的作业内容存放在 `body` 文件夹中。
这些作业文件通过 `\input` 命令或者 `\include` 命令被 `thesis.tex` 引用，你不需要单独编译它们。当你编译 `thesis.tex` 时，作业内容会自动整合到最终的PDF中。

### 修改作业内容

如果你需要修改某个作业：
1. 在文件列表中找到 `body` 文件夹并展开
2. 点击打开对应的作业文件（如 `HW1.tex`）
3. 修改内容后，重新编译 `thesis.tex` 即可看到更新后的PDF。
4. 记得更改 `thesis.tex` 中的引用语句，确保指向新的作业文件。同时`cover.tex`中的部分语句可能也需要更改。

### 常见问题

**Q: 编译时报错怎么办？**
- A: 确保打开的是 `thesis.tex` 文件，并且项目中的所有文件都完整
- A: 如果提示缺少包，Overleaf会自动安装；本地环境需要手动安装缺少的包
- A: 也可以考虑问问万能的AI大人

**Q: 为什么PDF中没有显示作业内容？**
- A: 请确认编译的是 `thesis.tex` 文件，而不是 `body` 中的某个作业文件
- A: 检查 `thesis.tex` 中是否包含了 `\input{body/HW1.tex}` 这样的引用语句
